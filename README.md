NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key_here

RESEND_API_KEY=your_resend_api_key_here 3. Run Development Server
Start the site locally to see your changes in real-time:

Bash

npm run dev
Open http://localhost:3000 in your browser.

âš™ï¸ Backend Setup (Free)
A. Setting up Reservations (Supabase)
Go to Supabase.com and create a free project.

Go to Project Settings > API to find your URL and ANON KEY. Paste these into your .env.local file.

Go to the SQL Editor in the left sidebar and paste this code to create your database table:

SQL

create table reservations (
id bigint generated by default as identity primary key,
created_at timestamp with time zone default timezone('utc'::text, now()) not null,
name text not null,
email text not null,
phone text not null,
date date not null,
time time not null,
guests int not null,
occasion text,
status text default 'pending'
);
Click Run. Your reservation form is now live!

B. Setting up Emails (Resend)
Go to Resend.com and create a free account.

Create an API Key and paste it into .env.local.

Important: In app/actions.js, look for the submitContactForm function. Change the to: email address to your own email to receive the messages.

ðŸŽ¨ How to Customize Content
You don't need to be a coder to change the text and images. 90% of the site's content is managed in one file.

1. Changing Text (Menu, FAQs, Reviews, etc.)
   Open app/data.js. You will see lists of data. simply change the text inside the quotes.

Example - Changing a Menu Item:

JavaScript

export const menuItems = [
{
id: 1,
name: "My New Steak Name", // Change this
category: "Steaks",
price: "$55.00", // Change price
desc: "Description of your new dish...",
},
// ...
]; 2. Changing Images
Go to the public/images/ folder.

Replace the existing images with your own.

Tip: Keep the file names the same (e.g., hero-wood.jpg) to avoid changing code. If you use new names, update the path in app/data.js or app/page.js.

3. Changing the "Secret Menu" Code
   Want a different code word?

Open components/SecretMenu.js.

Find this line:

JavaScript

const secretCode = "SHADOW";
Change "SHADOW" to your restaurant name or any code (e.g., "VIP2026").

4. Changing Colors
   The site uses Tailwind CSS.

Open app/globals.css.

The "Gold" accent color is #FFD700. You can Find & Replace this hex code throughout the project to match your brand color.

The easiest way to put this site online is with Vercel.

Click Deploy.
